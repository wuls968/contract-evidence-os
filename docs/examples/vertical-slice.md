# Vertical Slice Example

Input: read a local requirements file, extract critical constraints, and return a structured
summary with evidence references.

Expected runtime:

1. Compile contract from the goal and attachment path.
2. Generate a two-node plan graph with a verification gate.
3. Execute a file retrieval tool through a role passport.
4. Convert file contents into source and extraction evidence nodes.
5. Run shadow verification to confirm required evidence coverage.
6. Produce a delivery packet with facts, inferences, risks, and evidence references.
